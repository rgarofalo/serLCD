serLCD¶

This module implements the driver for LCD controller SparkFun Serial Enabled LCD Backpack. It implements most all functions from LiquidCrystal and added a few functions of my own http://www.sparkfun.com/datasheets/LCD/SerLCD_V2_5.PDF

class serLCD(drivername)¶

Creates a serLCD instance for controlling a diplay with 2 lines and 16 character through a serial port(es. SERIAL1). The SerLCD currently supports 16 and 20 character wide screens with 2 or 4 lines of display configurable using the appropriate functions.

Example:

import streams
import serLCD

error=streams.serial()
try:

    lcd=serLCD.lcd(SERIAL1)
    lcd.clear()
    lcd.home()
    lcd.setType(2,16)
    lcd.writeLine(0,'1234567890')

except Exception as e:
    print(e,stream=error)
